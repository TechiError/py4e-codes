# Python For Everybody Codes Coursera

This repository contains the code examples and exercises from the "Python for Everybody" Coursera specialization by Dr. Charles R. Severance. The code is organized by course and chapter for easy navigation.

## Repository Structure
- `course1/`: Code from the first course in the specialization.
- `course2/`: Code from the second course in the specialization.
- `course3/`: Code from the third course in the specialization.
- `course4/`: Code from the fourth course in the specialization.
    - `geodata/`: Contains geolocation data processing scripts.

## Getting Started
To run the code examples, ensure you have Python installed on your machine. You can clone this repository or [download](https://github.com/techierror/py4e-codes/archive/refs/heads/main.zip) the files directly.

```bash
git clone https://github.com/techierror/py4e-codes.git
cd py4e-codes
```
## Requirements
Some scripts may require additional libraries. You can install them using pip:

```bash
pip install -r requirements.txt
```
## Usage
Navigate to the desired course directory and run the Python scripts as needed. For example:

```bash
python course4/geodata/geoload.py
python course4/geodata/geodump.py
```
